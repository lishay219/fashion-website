# 👗 Rivo Fashion Website

### Modern Fashion Landing Page

A clean and responsive fashion website built with **HTML5** and **CSS3** to practice modern frontend development and responsive design techniques.

🔗 **Live Demo:** https://lishay219.github.io/fashion-website/

📂 **Repository:** https://github.com/lishay219/fashion-website

* Modern Hero Section
* Fashion Collection Showcase
* Customer Testimonials
* Newsletter Subscription
* Responsive Design
* Mobile Friendly Layout
* Clean UI & UX
* Elegant Footer Section
